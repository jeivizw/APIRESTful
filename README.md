# API RESTful
Para começar, eu ja fiz o trabalho de colocar o node_modules no arquivo para facilitar tudo.
# POREM
vou falar como se não estivesse presente no arquivo.

# Primeiro Passo
ao baixar os arquivos, descompacte-o e o abra no VS code

# Segundo Passo
após abrir no VS code, abra o terminal do VS code e digite duas coisas, primeiro "cd Backend" e depois "npm init -y"

# Terceiro Passo
após a conclusão da operação anterior, digite "npm install"

# Quarto Passo
depois disso, digite "npm install cors dotenv @supabase/supabase-js" (LEMBRANDO, tudo isso no terminal do vs code)

# Quinto Passo
seguindo como antes, digite "npm install nodemon" após a conclusão da operação.

# ULTIMO PASSO
Para fazer o projeto rodar, use o comendo "node server.js" para iniciar o servidor (como n tem um front-end, não tera uma super tela, só o aviso q a API esta rodando)
